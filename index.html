How to Use Golf Range Finder

Getting Started

Welcome to Golf Range Finder! This app helps you measure distances to holes on golf courses.

Using Existing Courses

You don't need to login to use existing courses:

  1. Select a course from the dropdown at the top
  2. Turn on GPS by clicking the toggle button
  3. Select a hole from the bottom panel
  4. The app will show your distance to the selected hole

Creating New Courses

To create your own courses:

  1. Sign up for an account or log in
  2. Navigate to the Dashboard using the button at the top
  3. Click "Create New Course" and follow the instructions
  4. Your courses will be available in the dropdown once approved

GPS Troubleshooting

If GPS isn't working:

  • Ensure location services are enabled on your device
  • Try refreshing the page
  • Make sure you're using a supported browser (Chrome, Safari, Firefox)
